# 基于混合整数线性规划的微电网多目标协同调度与寿命损耗优化研究

## 摘要

随着分布式可再生能源与电动汽车的大规模接入，现代园区微电网正向“源-网-荷-储-车”高度协同的模式演进。针对园区微电网在多能互补与设备健康管理下面临的复杂决策问题，本文基于混合整数线性规划（MILP）方法，构建了微电网协同调度与多目标优化模型，给出了精细化的设备运行方案，并深入揭示了分时电价套利与电池全生命周期健康之间的量化权衡规律。

**针对第一阶段问题一（确定性协同调度模型）**，本文以微电网系统总运行成本最小化为目标，统筹光伏消纳、固定储能（ESS）、电动汽车（EV）及可调建筑负荷，构建了确定性 MILP 协同调度模型。模型给出了全周期的精细化协同调度结果：

* **外网购电与光伏消纳**：系统实现了 100% 的光伏零弃光率（本地消纳 22969.3 kWh）。为最大化电价套利，模型在夜间谷电时段集中进行深度“填谷”购电，使得单时段峰值购电功率达到 863.7 kW，全周总购电量达 6.52 万 kWh。
* **储能与柔性负荷协同**：在用电高峰期，系统精准控制固定储能放电（全周总放电 456.0 kWh）、引导 EV 参与 V2G 反向供电，并配合可调负荷的时移，实现了对电网高价峰电的有效替代。 
* **优化成效**：对比非协同基准场景（总费用 46304.53 元），协同调度方案在满足所有建筑正常运行与 EV 离站需求（满足率 98.04%）的前提下，将总运行成本大幅压降至 38973.22 元，经济效益显著提升高达 15.83%，实现了园区柔性资源与电网价格信号的深度耦合。

**针对第一阶段问题二（兼顾寿命损耗的调度策略）**，本文进一步引入基于等效吞吐量的电池线性退化成本模型，将 ESS 与 EV 的循环寿命损耗折算为经济惩罚项，构建了兼顾短期运行成本与长期资产健康的双目标优化模型。通过改变寿命惩罚权重系数（$$\lambda$$），本文绘制了系统的帕累托前沿（Pareto Frontier），并得到了以下核心调度演变结果：

* **非对称边际转换效益**：在推荐折中决策点（$$\lambda=1$$）下，系统相较于纯经济驱动模式（$$\lambda=0$$），仅牺牲了 11.11 元的微小电费成本（增幅 0.029%），便成功削减了高达 186.57 kWh 的电池无效吞吐量（降幅 2.23%），极大地延长了电池使用寿命。
* **“阶梯式分工”与“套利死区”机制**：引入寿命惩罚后，模型自发调整了设备分工比例。由于 EV 动力电池的退化成本更高，系统优先削减了 EV 的充放电吞吐量（下降 173.92 kWh），而 ESS 吞吐量仅微降（12.65 kWh）。当峰谷价差收益不足以弥补物理折旧时，电池主动进入“套利死区”保持待机。最终形成了**“ESS 作为日常削峰填谷主力，EV 作为高价值应急备用”**的多元资源阶梯式调度格局。

综上所述，本文提出的协同调度模型不仅给出了切实可行的微电网物理运行方案，大幅提升了短期经济性，更通过精准的寿命损耗量化，实现了微电网全生命周期综合效益的帕累托最优，为现代智慧零碳园区的精细化运营提供了坚实的理论依据与决策支撑。
