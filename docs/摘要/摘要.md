摘要：随着分布式光伏、固定储能与电动汽车的大规模接入，新能源园区微电网正由传统供电模式向“源—荷—储—车”协同运行模式演进。针对园区在运行经济性、资源协调与电池健康管理方面面临的复杂决策问题，本文基于混合整数线性规划方法，构建了微电网—电动车—建筑系统的协同调度模型，并在此基础上研究了考虑寿命损耗的多目标优化策略。

针对问题一，本文建立了不考虑预测误差的确定性协同调度模型，以全周期运行费用最小为目标，统一优化外网购售电、固定储能充放电、电动车聚合充放电、建筑柔性调节及光伏消纳等决策。结果表明，在全周零弃光的条件下，协同调度相较于规则型非协同策略，可将系统运行费用由 46304.53 元降低至 38973.22 元，降幅约为 15.83%，说明多资源协同优化能够显著提升园区综合运行经济性。

针对问题二，本文进一步引入基于等效吞吐量的固定储能与电动车电池线性寿命损耗成本模型，构建兼顾短期运行费用与长期电池健康的协同优化模型，并通过权重扫描分析经济收益与寿命损耗之间的折中关系。结果表明，当寿命权重由 0 增加至 1 时，系统运行费用仅增加约 11.1 元，而固定储能与电动车总等效吞吐量下降约 186.6 kWh，其中吞吐收缩主要发生在电动车侧，说明寿命损耗约束能够有效抑制车队资源的高频调用，并促使固定储能承担更多替代性调节任务，系统由此表现出一定的分层协同特征。

研究结果表明，所建模型能够较好刻画新能源园区中源—荷—储—车多资源协同运行机制，实现运行经济性与电池健康性的合理平衡，可为园区综合能源系统的精细化调度提供参考。
