## 来源 1：V2G 优化中电池退化模型比较（加权安时吞吐 + 常数退化成本因子）

| 字段 | 内容 |
|------|------|
| **标题** | Assessing the incorporation of battery degradation in vehicle-to-grid optimization models |
| **作者 / 机构** | Valentin Preis, Florian Biedenbach；Forschungsstelle für Energiewirtschaft e.V. (FfE)，德国 |
| **年份** | 2023 |
| **链接** | https://doi.org/10.1186/s42162-023-00288-x （Springer *Energy Informatics*，开放获取） |
| **主要结论** | ① 系统综述后认为适合嵌入 V2G **线性优化**的策略主要有两类：**加权安时吞吐模型（wAh）+ 常数退化成本因子**，以及将退化与可测容量损失挂钩的 **performance-based（pb）** 模型。② 在其实现与情景下，**wAh 模型更适合作为目标函数中的退化项**（pb 模型在寿命初期 SoH 假设敏感，易导致退化成本被高估）。③ 退化成本按能量吞吐归一后，基线情景下 **pb 模型平均约 0.45 euro-cent/kWh**、**wAh 模型平均约 0.23 euro-cent/kWh**（文中对「cent/kWh」有明确定义：各优化步退化费用除以该步总能量吞吐）。④ 线性部分采用 **投资分摊 / 至 EOL 的最大能量吞吐** 思路：\(CF_{Deg}=Inv_{Bat}/E_{Bat\text{-}EoL}\)，与「更换（或资本）成本 ÷ 寿命期总吞吐」一致。⑤ 电芯层面循环老化基于 **Naumann et al. (2020)** 实验拟合；电池包投资参考 **BloombergNEF 2021 调查约 118 USD/kWh**，EOL 取 **容量损失 20%**（与 IEEE 1188 等工程惯例讨论一致）。 |
| **可提取参数** | 常数退化成本因子量级：**0.23–0.45 c€/kWh**（按文中基线平均）；快充（50 kW）情景 wAh 约 **0.36 c€/kWh**、pb 约 **0.7 c€/kWh**；电池包投资 **118 USD/kWh**（2021 BNEF，文中引用）；EOL **20%** 容量损失阈值。 |
| **是否适合论文引用** | **是**。开放获取期刊论文，DOI 稳定，方法段落与公式（如 Eq. 24–26）可直接支撑「线性吞吐量 + 常数单价」形式。 |
| **是否适合本题参数设定** | **是（尤其支撑 EV / V2G 侧数量级与模型形式）**。注意：文中欧元分/kWh 数值来自**特定电芯、驾驶曲线与电价套利情景**，不宜作为全国唯一标定；**适合作为下限参考 + 敏感性上界由工程溢价给出**（见 `problem2_battery_param_recommendation.md`）。 |

## 综合说明（与第二题建模对齐）

1. **线性吞吐量模型**：Preis & Biedenbach (2023) 的 **wAh 模型 + 常数 CF~€/kWh**、Xu (2017) 的**分段线性循环成本**、NREL REopt 的**吞吐—退化联立**，均从不同角度支持在优化中使用**可分离、凸或线性代理**的退化成本项；Gailani et al. (2020) 提醒**模型误差需敏感性覆盖**。  
2. **ESS 与 EV 不同系数**：Preis 等明确 **EV 包投资用 BNEF 乘用车 pack 价**与 **V2G 专用循环曲线**；固定储能工程上常用**独立质保与循环目标**，与车载电池在**日历老化、热管理目标、残值与车主偏好**上不同——文献与工程惯例均支持**分别标定**（见推荐与敏感性文档）。  
3. **「更换成本 + 循环寿命 + 容量」折算**：Preis 文式 **\(CF_{Deg}=Inv_{Bat}/E_{Bat\text{-}EoL}\)** 即典型 **资本成本 / 寿命期总允许吞吐（kWh）** 思路，与数模中「元/kWh」线性项一致。
