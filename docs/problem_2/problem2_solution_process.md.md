### 3. 模型求解过程与算法设计

考虑到本模型包含大量的连续决策变量（各设备充放电功率、购售电功率等）以及 0-1 整数决策变量（充放电互斥状态），该问题属于典型的**混合整数线性规划（MILP）问题**。为了高效、稳定地获取全局最优解并进行多目标权衡分析，本文设计了如下求解过程：

#### 3.1 数据预处理与异质性参数映射
在构建数学模型前，首先对输入数据进行清洗与降维处理：
1. **EV 状态稀疏化**：根据电动汽车的到离站时间，仅在车辆实际在站的时段内创建充放电决策变量，大幅削减了模型维度。
2. **异质性参数映射**：基于前文对 EV 数据的深度挖掘，将不同车型（如 SUV、轿车、紧凑型）的单位吞吐退化成本（$$c_{deg, i}^{ev}$$）精准映射至对应的车辆会话中，为后续的精细化调度提供参数基础。

#### 3.2 综合目标函数与约束矩阵构建
基于 Python 环境下的 PuLP 建模框架，将微电网的物理运行逻辑转化为计算机可识别的数学矩阵：
1. **目标函数组装**：将纯运行成本（购售电、碳排放、弃光惩罚、柔性负荷调节）与电池寿命损耗成本（ESS 与 EV 退化成本）进行线性加权求和。
2. **约束条件注入**：严格注入瞬时功率平衡、储能/EV 荷电状态（SOC）时序递推、充放电互斥（引入 0-1 变量）、以及建筑柔性负荷的平移与恢复动力学约束。特别地，引入了 ESS 终端 SOC 模式（严格等于或宽松大于初值）与 EV 最小安全 SOC 比例约束，以保障工程实用性。

#### 3.3 基于分支定界法的 MILP 求解
调用高性能开源求解器 CBC（Coin-or branch and cut）对 MILP 模型进行全局寻优。
* **算法原理**：求解器底层采用**分支定界法（Branch and Bound）**结合**割平面法（Cutting Planes）**。首先松弛整数约束求解线性规划问题，随后通过不断添加有效割平面和分支搜索，逐步逼近最优整数解。
* **求解参数设置**：为平衡求解精度与计算时间，设定最大求解时间限制（Time Limit）为 600 秒，相对最优间隙（Relative Gap）容忍度设为 1%。

#### 3.4 多目标权衡的帕累托扫描（Weight Scanning）
为了量化分析短期经济收益与长期电池健康之间的折算权衡，本文设计了自动化的权重扫描算法：
1. 设定退化成本权重系数 $$\lambda$$（对应代码中的 `ess_deg_weight` 与 `ev_deg_weight`）的扫描区间与步长。
2. 在外层循环中，依次更新目标函数中的 $$\lambda$$ 值，并重复调用 CBC 求解器进行独立求解。
3. 记录每次求解的最优纯运行成本与系统总吞吐电量，从而在二维目标空间中拟合出**帕累托前沿曲线（Pareto Frontier）**。

#### 3.5 结果解耦与多维指标提取
在获取最优解后，算法自动进行多维度的后处理分析：
* **成本对账（Objective Breakdown）**：将总目标函数值严格解耦为购电成本、售电收益、弃光惩罚、各项退化成本等独立分项，验证求解的一致性。
* **物理指标统计（Operational Metrics）**：计算全周期的电网交互电量、弃光电量、以及 ESS 和 EV 的总吞吐电量。
* **车型行为汇总（EV Type Summary）**：按车型聚合统计 EV 的 V2B 参与度与放电量，以验证模型是否自发形成了预期的“阶梯式分工”机制。
