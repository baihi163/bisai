# 第二题模型求解过程（论文版）

下面在问题一模型基础上，进一步考虑固定储能与电动车电池在频繁充放电过程中的寿命损耗，并据此重新优化园区协同调度策略。与第一问相比，第二问的重点不再只是降低短期运行成本，而是通过引入电池退化成本，建立短期经济收益与长期设备健康损耗之间的统一权衡机制。结合当前项目的真实实现，问题二采用与问题一一致的 7 天、15 min 分辨率时域离散方式，共得到 `672` 个调度时段，并在相同功率平衡框架下完成寿命感知调度求解。

---

## 一、寿命损耗指标与寿命成本模型的构造

为回答题目中“为固定储能和电动车构造可计算的寿命损耗指标或寿命成本模型”的要求，本文采用**基于能量吞吐量的线性寿命成本模型**。该模型的优点在于：一方面能够反映“充放电越频繁、总吞吐越大，则寿命损耗越高”的基本物理规律；另一方面保持了目标函数的线性结构，便于直接嵌入现有优化框架并由数学规划求解器稳定求解。

对于固定储能，设第 `t` 个时段的充电功率和放电功率分别为 `P_ESS_ch(t)` 和 `P_ESS_dis(t)`，时间步长为 `Δt = 0.25 h`，则固定储能的寿命损耗成本可表示为：

`C_ESS_deg = sum( w_ESS * c_ESS * (P_ESS_ch(t) + P_ESS_dis(t)) / 2 * Δt )`

其中，`c_ESS` 表示单位吞吐电量对应的寿命成本，`w_ESS` 为固定储能寿命损耗权重。

对于电动车聚合体，采用相同思想构造寿命成本模型。若第 `i` 辆电动车在可参与调度时段集合 `T_i` 内的充放电功率分别为 `P_EV_ch(i,t)` 和 `P_EV_dis(i,t)`，则其寿命损耗成本可表示为：

`C_EV_deg = sum_i sum_t [ w_EV * c_EV(i) * (P_EV_ch(i,t) + P_EV_dis(i,t)) / 2 * Δt ]`

其中，`c_EV(i)` 为第 `i` 辆车单位吞吐电量对应的寿命成本，`w_EV` 为电动车寿命损耗权重。

在参数来源上，固定储能的单位吞吐退化成本由设备参数文件中的电池退化成本字段提供，电动车的单位退化成本由会话数据链中的退化成本字段给出；若题目未直接给出相关参数，则可采用“设备购置成本—额定容量—循环寿命”折算方式，将总寿命价值折算为单位吞吐电量成本。该处理方式既满足题目“构造可计算寿命损耗指标”的要求，也与当前项目代码中的 `p2.py` 和 `p_1_5_ultimate.py` 实现保持一致。

## 二、考虑寿命损耗后的重新优化调度策略

在寿命成本模型构建完成后，问题二在问题一协同优化模型基础上保留原有调度变量，包括外网购售电功率、固定储能充放电功率、电动车聚合充放电功率、建筑柔性负荷调节量以及必要时的弃光功率等，并在目标函数中加入固定储能与电动车电池寿命损耗成本项，从而形成新的综合优化目标。此时目标函数可理解为“运行成本 + 电池健康成本”的综合最小化。

由于寿命成本项仍为线性形式，因此问题二模型与问题一一样，可转化为线性规划或混合整数线性规划问题，并采用相同求解框架进行全周期联合优化。求解得到的各时段最优变量，包括外网购电功率 `P_buy(t)`、固定储能充放电功率 `P_ESS_ch(t)`、`P_ESS_dis(t)`、电动车聚合充放电功率 `P_EV_ch(t)`、`P_EV_dis(t)`、建筑移位与恢复负荷以及弃光功率等，共同构成了**重新优化后的调度策略**。

结合统一口径下的全周结果可知，当 `w_ESS = w_EV = 1` 时，问题二与问题一主结果数值一致，即问题二在该权重下退化为问题一的基准解；因此，第二问的“重新优化”重点体现在寿命权重变化所引发的策略调整，而不在于重新定义变量集合。为进一步体现这一过程，本文对寿命权重进行了敏感性扫描，分别考察 `w = 0, 0.1, 0.5, 1, 2` 等情形下系统调度行为与综合成本的变化。

## 三、引入寿命损耗前后调度行为的变化

为回答题目中“比较引入电池寿命损耗前后系统调度行为的变化”的要求，本文采用统一时域和统一数据口径，对问题一与问题二进行了对比分析。

其中，问题一选取 `p1_ultimate_latest` 的全周结果，问题二选取全周权重扫描结果 `scan_auto_weight_scan/w_1/` 作为与问题一严格可比的基准场景，同时进一步选取 `w = 0` 和 `w = 2` 作为寿命损耗弱化与强化的对照场景。

在全周主场景下，问题一与问题二在 `w = 1` 时完全一致，总目标值约为 `39467.46 元`，其中运行成本约为 `38973.22 元`，固定储能寿命损耗成本约为 `369.39 元`，电动车寿命损耗成本约为 `124.85 元`，外网总购电量约为 `65263.90 kWh`。固定储能全周吞吐量约为 `6716.19 kWh`，电动车全周吞吐量约为 `1441.38 kWh`，系统峰值购电功率约为 `863.7 kW`。

进一步比较不同寿命权重下的结果可知，当寿命权重从 `0` 增加到 `2` 时，系统的运行成本由约 `38962.11 元` 上升至 `38973.54 元`，而固定储能和电动车的寿命损耗成本显著增加，表明优化器开始更加重视电池健康价值；与此同时，电动车放电量由 `562.245 kWh` 降至 `392.415 kWh`，电动车吞吐量由 `1615.30 kWh` 降至 `1436.30 kWh`，说明在寿命损耗权重提高后，模型显著抑制了电动车的高频参与和深度放电行为。固定储能吞吐量则基本保持在 `6716 kWh` 左右，变化幅度较小，说明在当前算例中，固定储能仍是承担调节任务的主力资源，而电动车更多作为辅助型灵活资源参与调度。

这说明：引入寿命损耗后，模型不会简单地追求更低电费，而是自动减少那些“收益不足以覆盖寿命代价”的充放电行为，从而使调度策略由“积极套利型”逐步转向“健康友好型”。

## 四、固定储能与电动车在削峰、储能和供能中的分工比例

为回答题目中“优化固定储能设备与电动车在削峰、储能和供能中的分工比例”的要求，本文进一步对两类资源的角色分工进行量化分析。考虑到仓库中未直接提供现成的“分工比例”指标，本文基于全周时序结果和运行指标进行后处理定义。

### 1. 储能分工比例

“储能分工”反映两类资源承担系统能量缓冲任务的相对份额。本文采用全周吞吐量占比进行定义：

- ESS 储能分工比例  
  = ESS 全周吞吐量 / (ESS 全周吞吐量 + EV 全周吞吐量)

- EV 储能分工比例  
  = EV 全周吞吐量 / (ESS 全周吞吐量 + EV 全周吞吐量)

在主场景 `w = 1` 下，固定储能吞吐量为 `6716.19 kWh`，电动车吞吐量为 `1441.38 kWh`，因此固定储能储能分工占比约为 `82.3%`，电动车约为 `17.7%`。这说明系统主要依赖固定储能承担能量吸纳与转移任务。

### 2. 供能分工比例

“供能分工”反映两类资源对负荷侧放电支撑的相对贡献。本文采用全周放电能量占比定义：

- ESS 供能分工比例  
  = ESS 全周放电量 / (ESS 全周放电量 + EV 全周放电量)

- EV 供能分工比例  
  = EV 全周放电量 / (ESS 全周放电量 + EV 全周放电量)

在主场景 `w = 1` 下，固定储能全周放电量为 `6372.0 kWh`，电动车全周放电量为 `397.235 kWh`，由此得到固定储能供能占比约为 `94.1%`，电动车供能占比约为 `5.9%`。可见在系统供能职责上，固定储能远高于电动车，是削峰补能的核心执行者。

### 3. 削峰分工比例

“削峰分工”主要体现两类资源在高价高负荷时段替代外网购电、缓解电网压力的能力。本文将购电价处于全周前 `20%` 分位以上的时段定义为峰时段，共得到 `144` 个峰时段；并以峰时段内的净放电能量作为削峰贡献度量。

在问题一主场景下，固定储能峰段净放电能量为 `6360.0 kWh`，电动车峰段净放电能量为 `381.301 kWh`；在寿命权重强化至 `w = 2` 时，固定储能峰段净放电能量保持在 `6360.0 kWh`，而电动车峰段净放电能量下降至 `376.482 kWh`。这说明随着寿命惩罚增强，电动车在高峰削峰任务中的参与程度进一步下降，而固定储能继续承担绝大多数峰段支撑任务。

因此，从削峰、储能和供能三个维度来看，第二题优化后系统形成了较清晰的分工结构：**固定储能负责主力削峰和主力供能，电动车承担补充型储能与辅助型供能角色**。随着寿命权重提高，这种分工进一步强化。

## 五、短期经济收益与长期电池健康损耗的折算与权衡

为回答题目中“分析短期经济收益与长期电池健康损耗之间应如何折算和权衡”的要求，本文将电池寿命损耗统一折算为与电费同量纲的货币成本，并直接纳入目标函数与运行成本共同优化。这样一来，每增加一次充放电行为，优化器都会同时考虑其带来的两类影响：一类是降低购电成本或减少弃光惩罚所带来的即时收益，另一类是增加电池寿命损耗成本所带来的长期代价。

从结果来看，当寿命权重为 `0` 时，系统等价于“只重视运行收益、忽略健康惩罚”，因此电动车放电量和吞吐量均显著较高；而当权重逐步增加至 `1` 和 `2` 时，模型会自动削减那些边际收益较小但寿命代价较高的电池动作，尤其是电动车的反向供电行为收缩明显。这说明寿命成本的引入确实改变了调度策略的价值判断机制。

在工程解释上，可以将某一充放电行为是否“值得”理解为：该行为带来的购电成本节约、弃光减少收益或峰时支撑收益，是否能够覆盖相应增加的寿命损耗成本。若边际收益低于边际寿命代价，则优化器会倾向于减少该行为。因此，问题二本质上是在求解一个“短期经济收益—长期设备健康”之间的最优折中点。

结合敏感性结果可知，`w = 1` 对应题目给定退化成本原始口径下的基准方案，适合作为正文主方案；`w = 0` 可作为“忽略寿命惩罚”的下界参考，`w = 2` 可作为“强化寿命保护”的上界情形，用于说明系统在不同健康偏好下的调度变化规律。由此，问题二不仅给出了重新优化后的调度结果，也清晰展示了短期经济性与长期健康性之间的折中关系。

---

## 六、小结

综上，第二题在问题一协同优化模型基础上，引入了固定储能与电动车电池的线性吞吐寿命成本模型，并通过寿命权重扫描实现了运行成本与设备健康损耗之间的统一权衡。求解结果表明：在基准权重下，问题二与问题一主结果一致；当寿命惩罚增强时，系统会主动减少电动车放电和高频循环行为，使固定储能承担更多削峰、供能和储能任务，从而形成“固定储能主导、电动车辅助”的分工格局。这说明所建模型能够有效回答题目关于寿命建模、重新优化、行为变化、分工比例以及经济—健康权衡的全部要求。
