## 四、固定储能与电动车在削峰、储能和供能中的分工比例

为回答题目中“优化固定储能设备与电动车在削峰、储能和供能中的分工比例”的要求，本文基于全周调度结果，对固定储能（ESS）与电动车（EV）的任务分担进行量化分析。考虑到原始结果文件中未直接给出“分工比例”字段，本文根据吞吐量、放电量和峰时净放电量进行后处理统计。

### 1. 储能分工比例

储能分工比例用全周吞吐量占比表示，即：

- ESS 储能分工比例 = `ess_throughput_kwh / (ess_throughput_kwh + ev_throughput_kwh)`
- EV 储能分工比例 = `ev_throughput_kwh / (ess_throughput_kwh + ev_throughput_kwh)`

在主场景 `w = 1` 下，固定储能吞吐量为 `6716.19 kWh`，电动车吞吐量为 `1441.38 kWh`，因此：

- ESS 储能分工占比约为 `82.3%`
- EV 储能分工占比约为 `17.7%`

这说明系统主要依赖固定储能承担能量吸纳与时移任务。

### 2. 供能分工比例

供能分工比例用全周放电能量占比表示，即：

- ESS 供能分工比例 = `ess_discharge_energy_kwh / (ess_discharge_energy_kwh + ev_discharge_energy_kwh)`
- EV 供能分工比例 = `ev_discharge_energy_kwh / (ess_discharge_energy_kwh + ev_discharge_energy_kwh)`

在主场景 `w = 1` 下，固定储能全周放电量为 `6372.0 kWh`，电动车全周放电量为 `397.235 kWh`，因此：

- ESS 供能分工占比约为 `94.1%`
- EV 供能分工占比约为 `5.9%`

可见在系统供能职责上，固定储能远高于电动车，是削峰补能的核心执行者。

### 3. 削峰分工比例

削峰分工比例主要反映两类资源在高价高负荷时段替代外网购电、缓解电网压力的能力。本文将购电价处于全周前 `20%` 分位以上的时段定义为峰时段，共得到 `144` 个峰时段，并以峰时段内的净放电能量作为削峰贡献度量。

其统计方法为：

- ESS 峰段净放电量  
  `sum((P_ess_dis - P_ess_ch) * delta_t, t in T_peak)`

- EV 峰段净放电量  
  `sum((P_ev_dis - P_ev_ch) * delta_t, t in T_peak)`

进而定义：

- ESS 削峰分工比例 = `ess_peak_energy / (ess_peak_energy + ev_peak_energy)`
- EV 削峰分工比例 = `ev_peak_energy / (ess_peak_energy + ev_peak_energy)`

在问题一主场景下，固定储能峰段净放电能量为 `6360.0 kWh`，电动车峰段净放电能量为 `381.301 kWh`；在寿命权重提高至 `w = 2` 时，固定储能峰段净放电能量保持在 `6360.0 kWh`，而电动车峰段净放电能量下降至 `376.482 kWh`。这说明随着寿命惩罚增强，电动车在高峰削峰任务中的参与程度进一步下降，而固定储能继续承担绝大多数峰段支撑任务。

因此，从削峰、储能和供能三个维度来看，第二题优化后系统形成了较清晰的分工结构：**固定储能负责主力削峰和主力供能，电动车承担补充型储能与辅助型供能角色**。随着寿命权重提高，这种分工进一步强化。
