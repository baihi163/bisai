# 问题一 baseline 模型建立（非协同规则基准调度）

## 4.1 问题一建模目标
本题研究对象为夏季科研园区综合能源系统：在建筑用电、分布式光伏、外部电网及园区电动汽车（EV）并存条件下，考察多主体在**缺乏协同优化**时的运行机理与系统表现。问题一不构造协同优化目标函数，而建立**非协同规则型基准调度（baseline）**模型，目的在于：

1. 在给定外部环境与设备参数下，得到与数据时间分辨率一致的可复现运行轨迹；
2. 通过功率平衡与能量状态递推，刻画“规则驱动”情形下光伏消纳、购电依赖、储能动作及 EV 补能过程的耦合关系；
3. 基于统一评价指标量化经济性、可靠性及资源利用水平，为后续研究提供**对照基准**并揭示进线受限、辐照波动等条件下的**潜在瓶颈**。

该目标属于**机理可解释的规则仿真**，而非以目标函数最优为核心的协同优化建模。

---

## 4.2 模型基本假设
1. **时间离散化**：调度步长与数据一致，采用均匀时间栅格，步长记为 $$\Delta t$$（如 15 min），全研究时段由索引集合 $$T$$ 表示。

2. **网络抽象**：园区等效为单母线结构，不计线路阻抗与网损，各元件功率在任一步长内以**有功功率瞬时平衡**描述。

3. **建筑侧基准**：建筑用电采用给定的**原生负荷曲线**，柔性调节在 baseline 中取**零**，即不实施移峰、削减等需求响应。

4. **EV 行为规则**：车辆在会话持续时间内仅按**到站即充**规则获取充电功率，不向园区反向放电（无车网互动放电）；充电效率采用常数 $$\eta_{ev}$$ 表征交流侧功率与电池能量增量的折算关系；离站能量未达下限时在结果中予以记录，不终止仿真。

5. **固定储能规则**：储能**不从电网充电**；充电功率仅来源于满足原生负荷与 EV 充电后的剩余光伏；放电仅当购电价格处于预先定义的**高价时段集合**且存在功率缺口时启动，并受额定功率与荷电状态（SOC）边界约束。高价时段由全研究时段购电价格的**分位数阈值**确定，属于**外生规则**而非优化结果。

6. **电网交换**：向电网购电、上网功率分别受**时变上限**约束；当购电已达上限仍无法满足净负荷时，允许出现**未满足功率缺口**并计入输出，以表征刚性受限下的供电不足。

7. **碳排放核算**：电网侧碳排放由购电量与时变电网碳强度乘积近似，不包含全生命周期展开。

8. **数据有效性**：输入会话与参数经完整性、边界一致性等校验；异常记录在案或按约定规则剔除，以保证仿真可执行。

---

## 4.3 系统组成与边界
物理组成包括：

- 三类建筑聚合原生负荷
- 可利用光伏（以各时段可用出力上限表征）
- 并网接口（分时购电价、上网电价、进/出口功率上限及可选的碳强度）
- 固定电化学储能
- 按会话到离站描述的 EV 群体
- 建筑柔性负荷能力（在 baseline 中调节量取零，边界上仍保留为系统能力描述）

系统边界取园区与外部电网的交换断面：

- 进入边界的能量流为**购电**
- 离开边界的能量流为**上网电量**
- 若采用碳指标，则可进一步核算电网购电对应的碳排放

EV 车载能量在到离站时刻与园区发生能量交换，会话外车辆不计入模型。

时间边界覆盖题目所给连续多日研究区间，所有外生序列（负荷、光伏上限、电价、限额等）在该区间内给定，不外延预测。

决策边界方面，baseline 中**不存在**以全局成本或碳排为目标的整体优化变量；各元件动作由**固定优先级与阈值规则**唯一确定，从而与协同优化模型在范式上区分。

## 4.4 输入输出定义

### 输入
主要输入可分为四类：

1. **时序外生变量**  
   环境温度、辐照（若使用）、各建筑及合计原生负荷、光伏可用功率上限、购售电价、电网边际碳强度（若使用）、购电与上网功率上限等。

2. **EV 会话参数**  
   到离站时刻、电池容量、到站能量、离站最低需求能量、最大充放电功率及是否允许放电等。

3. **固定资产参数**  
   储能额定能量与功率、充放电效率、SOC 允许区间、初始能量；光伏与充电设施相关额定或并发能力参数（若建模中显式使用）。

4. **柔性负荷参数**  
   在 baseline 中不参与决策，可作为扩展或灵敏度分析输入。

### 输出
主要输出包括：

1. **逐时段功率与能量轨迹**  
   原生负荷、EV 总充电功率（交流侧）、含 EV 总需求、光伏本地消纳、光伏对储能充电、上网与弃光、储能能量、储能充放电功率、购电功率、上网功率、储能动作前后净负荷等。

2. **可靠性表征量**  
   购电达上限后的未满足功率，以及由积分得到的未供电量。

3. **汇总指标**  
   购电成本与上网收益、购电碳排放、光伏利用率或弃光率、EV 离站能量达标程度的全周统计量等。

输入输出之间通过**功率平衡与状态方程**联结；符号表与量纲约定宜在全文统一列出，以避免交流侧与直流侧（电池侧）能量混淆。

## 4.5 baseline 调度思想
非协同 baseline 的核心是：

**“局部规则 + 固定次序”**，而非全局最优。

### 1. 需求侧
- 建筑按给定曲线取电；
- EV 在停留期间按补能需求与功率上限确定充电功率；
- EV 不响应电价或光伏余量；
- EV 不向母线放电。

### 2. 供给侧优先级
- 光伏优先满足原生负荷与 EV 充电；
- 剩余光伏按规则用于储能充电；
- 再考虑上网（受出口上限约束）；
- 超出部分形成弃光。

### 3. 储能与电网的衔接
在每一时段：

- 先形成光伏就地消纳后的净需求；
- 若该时段属于高价时段且净需求为正，则储能在允许范围内放电削峰；
- 随后由电网购电填补剩余缺口；
- 购电受进口上限约束；
- 若仍有正缺口，则记为未满足负荷。

### 4. 时段内计算次序
与上述思想一致，每个时间步按如下顺序执行：

1. 确定 EV 充电功率，得到总需求；
2. 执行光伏本地分配；
3. 在满足高价规则时对净需求进行储能放电；
4. 由电网购电补足剩余缺口；
5. 将剩余光伏用于储能充电、上网或弃光。

该机制体现了工程可实施的简化策略：储能不承担套利优化，仅在高价缺电时释放能量；光伏优先本地消纳，有利于直观解释弃光、上网与储能充电之间的竞合关系。

需注意：若多时段内本地需求持续高于光伏可用上限，则不存在剩余光伏对储能充电，储能将主要依赖前期充电积累在高价缺电时段放电，直至 SOC 触及下界，此为规则与数据共同作用的结果。

## 4.6 模型适用性说明
本 baseline 模型适用于：

- 赛题要求在协同优化之前建立可对照、可复现的“非协同运行基准”；
- 强调运行机理透明、便于图示化能量流的论文叙事；
- 数据提供 15 min（或类似）级外生序列与会话级 EV 描述，需与实测或给定曲线逐点对照的验证场景。

不适用或需谨慎外推的情形包括：

1. **严格零失负荷场景**  
   若要求严禁失负荷且不允许任何松弛量，则需改写购电缺口处理机制。

2. **最优调度问题**  
   若研究目标是最优购售电、移峰填谷或车网协同定价，则应采用优化模型而非本规则 baseline。

3. **更高保真电力系统分析**  
   若需考虑潮流、电压、三相不平衡或动态过程，应扩展为更细致的电力系统模型。

此外，未显式刻画充电桩同时接入数等站端并发约束时，模型给出的 EV 功率为会话功率上限意义下的上界情景，实际同时率可能使结果偏乐观，宜在适用性讨论中说明。

## 建议补充说明
1. **未满足功率的物理含义**  
   建议进一步明确其在工程上与“切负荷”或“停电”的对应关系，并与符号定义严格一致。

2. **EV 与储能效率设定**  
   常数效率是对充电机与变流器的聚合近似；若论文可补充敏感性分析，将增强模型稳健性表达。

3. **高价时段分位数阈值**  
   例如 0.80 分位对应“最贵 20% 时段”，应在文中解释其物理意义及退化情形，避免误解为优化结果。

4. **柔性负荷为零的含义**  
   这属于 baseline 的建模界定，而不表示园区本身不存在柔性调节能力。

5. **赛题原文缺失的风险说明**  
   当前“问题一”边界按非协同基准理解；若正式赛题第一问另含协同优化或不同失负荷约定，应以官方原文为准调整本节表述，并保持全文符号一致。
