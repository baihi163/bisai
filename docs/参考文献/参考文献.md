# 1 文献综述

随着分布式光伏、固定储能系统以及电动汽车的大规模接入，园区级综合能源系统正在由传统的单向供电模式向“源—荷—储—车”协同运行模式转变。对于具备光伏发电、固定储能、电动车充放电设施及多类建筑负荷的现代园区而言，如何在满足建筑运行需求与车辆出行需求的前提下，提高可再生能源消纳水平、降低运行成本与碳排放，并兼顾电池寿命损耗，已成为微电网优化调度领域的重要研究方向。已有研究主要围绕微电网协同调度、含电动汽车的能量管理、建筑柔性负荷参与优化以及电池寿命损耗建模等方面展开。

## 1.1 微电网与园区综合能源系统协同调度研究

微电网能量管理系统的核心任务是协调分布式电源、储能设备、可控负荷以及外部电网之间的功率分配关系，从而实现经济、安全、低碳的运行目标。Olivares 等对微电网控制与调度的发展趋势进行了系统综述，指出微电网研究正在从单一设备控制逐步转向多资源、多时间尺度协同优化框架。Shekari 等针对多载体微电网提出了基于混合整数线性规划的能量管理方法，为含多类能源设备的统一优化建模提供了可借鉴的数学框架。Manshadi 和 Khodayar 则从韧性运行的角度讨论了多能流微电网的协调问题，说明在复杂运行条件下，统一调度框架不仅关系到经济性，也关系到系统抗扰动能力。

近年来，园区综合能源系统的研究进一步强调多场景、多时间尺度和需求响应机制的结合。部分文献已经开始将园区负荷特征、可再生能源出力和用户侧响应行为纳入同一优化模型中，从而更贴近实际运行场景。然而，从现有研究看，多数工作仍聚焦于“源—荷—储”中的部分环节，对建筑负荷柔性、电动车集群接入以及储能退化的统一刻画仍不充分。尤其在园区边界下，将光伏、固定储能、EV 与建筑柔性负荷同时纳入一个可解、可解释的调度模型，仍具有较强的研究价值。

## 1.2 含电动汽车的微电网优化调度研究

随着电动汽车保有量快速增长，EV 不再只是新增负荷，也逐渐被视为具有灵活调节潜力的移动储能资源。相关研究表明，通过有序充电或车网互动（V2G）机制，EV 可以在削峰填谷、缓解购电压力和提升可再生能源消纳方面发挥积极作用。Shekari 等研究了电动汽车车队参与微电网能量管理的技术经济协同问题，说明 EV 集群在微电网中具有重要的调度价值。Van Roy 等则围绕办公楼微电网场景分析了分布式能源与电动汽车充电的协同关系，为“建筑—车—微网”联合优化提供了较为典型的应用背景。

不过，现有 EV 相关研究也存在较明显的局限。首先，一部分研究采用单用户或单车辆模型，难以反映园区停车场中电动车大规模聚合接入的运行特征；另一部分车队模型则往往依赖统计平均假设，对到离站时间、初始荷电状态和离站需求等异质性约束刻画不足。其次，部分研究虽然考虑了 V2G 带来的经济收益，但未充分讨论 EV 电池频繁充放电对健康状态和用户接受度的影响。因此，在园区微电网场景下，如何在满足车辆使用需求的同时，将 EV 作为可调节资源纳入统一调度，并进一步讨论其与固定储能之间的功能分工，仍是值得深入研究的问题。

## 1.3 建筑柔性负荷与需求响应参与调度研究

建筑负荷是园区综合能源系统中的重要组成部分。与传统刚性负荷不同，部分建筑负荷在时间上具有一定的可调节性，例如可转移负荷、可削减负荷以及受舒适度约束的空调类负荷等。通过合理设计需求响应机制，可以在不显著影响用户体验的前提下，实现负荷平滑、峰值削减与运行成本优化。Fontenot 和 Dong 对建筑集成微电网的建模、控制与能量管理进行了综述，指出建筑侧柔性在提升系统整体调节能力方面具有重要作用。相关研究还从分层控制、多时间尺度调度以及激励型需求响应等角度，对建筑负荷参与园区优化的方式进行了拓展。

尽管建筑柔性负荷已经成为综合能源系统研究中的重要方向，但现有研究仍存在两方面不足。一方面，不少文献重点关注建筑侧热惯性、用户满意度或局部需求响应机制，而与电动汽车、固定储能等其他柔性资源的联合建模相对较少；另一方面，建筑柔性常与启发式算法、双目标优化或控制策略研究相结合，在与可解释的 MILP 调度框架衔接时仍存在一定鸿沟。因此，在园区微电网中，将建筑柔性与光伏、储能和 EV 协同建模，有助于更全面地反映“源—荷—储—车”之间的耦合关系。

## 1.4 基于 MILP 的微电网调度建模研究

在微电网与综合能源系统调度问题中，混合整数线性规划因其建模清晰、约束表达能力强以及在适当线性化条件下可获得全局最优解等优点，被广泛应用于设备启停、储能充放电互斥、购售电决策以及需求响应安排等问题。Shekari 等提出的多载体微电网 MILP 模型表明，该方法能够较好地处理多类能源设备之间的功率平衡与运行约束。Shao 等从多载体系统优化潮流的角度说明了线性化优化方法在复杂能量网络中的适用性。Cardoso 等、Oliveira 等的研究进一步表明，MILP 框架不仅适用于传统经济调度，也能够在一定程度上融合储能老化或退化成本建模。

当然，MILP 的适用也伴随着一定代价。例如，储能退化、电池寿命、交流潮流等特性本身具有明显非线性，若要嵌入 MILP 往往需要采用线性代理、分段近似或附加二进制变量，这会增加模型维度和求解复杂度。此外，在多资源、大规模时序调度问题中，二进制互斥变量过多也可能显著增加计算负担。尽管如此，相比于纯启发式算法，MILP 在模型可解释性、约束可控性以及工程复现性方面仍具有明显优势，因此仍是园区协同调度研究中的重要建模路径。

## 1.5 电池寿命损耗与经济性—健康性权衡研究

在储能系统和电动汽车参与调度的场景中，电池寿命损耗问题越来越受到关注。若调度模型仅以短期购电成本最小为目标，往往会倾向于频繁调用储能或 V2G 功能，从而造成电池吞吐量增加和寿命加速衰减。因此，将电池退化成本纳入优化目标，是实现调度结果工程可持续性的重要前提。Ortega-Vazquez 较早在 EV 充放电调度中显式考虑了电池退化与价格不确定性，说明是否计入退化成本会显著影响车辆最优充放电策略。随后，Seydenschwanz 等尝试将循环老化进行线性化处理并嵌入 MILP 模型，Cardoso 等与 Oliveira 等则在微电网规划或运行层面进一步讨论了储能老化与经济调度的耦合关系。

总体来看，现有研究在“电池退化建模”方面已经取得一定进展，但仍存在几个值得关注的问题。首先，许多研究仅关注固定储能退化，而对 EV 电池健康的刻画相对简化；其次，不同研究在退化模型复杂程度上差异较大，从简单吞吐量代理到基于循环深度、温度、SOC 区间的复杂模型不等，但复杂模型往往难以直接融入可求解的优化框架；再次，虽然多目标优化研究较多，但真正能够清晰展示“短期经济收益—长期寿命损耗”量化权衡过程的工作仍然有限。基于此，在园区调度问题中同时考虑 ESS 与 EV 的寿命损耗，并通过统一目标函数或权重机制分析二者在削峰、储能和供能中的分工关系，具有较强的现实意义和方法价值。

## 1.6 现有研究评述与本文切入点

综上所述，现有文献已在微电网协同调度、EV 参与优化、建筑柔性负荷管理、MILP 建模以及电池寿命损耗建模等方面奠定了较好的研究基础，但仍存在以下不足。

1. **资源协同建模仍较碎片化。** 不少研究只关注固定储能、需求响应或 EV 中的单一资源，缺少对光伏、固定储能、电动汽车与建筑柔性负荷的统一调度框架。
2. **EV 与 ESS 的分工机制研究不足。** 已有文献往往将 EV 简化为可调负荷或附加储能，较少系统讨论固定储能与移动储能在不同目标权重下的角色差异。
3. **寿命损耗与经济性权衡仍不充分。** 虽然已有研究考虑了电池退化成本，但多数仅针对单类电池或单一场景，缺少在园区级协同调度中同时讨论 ESS 与 EV 退化的统一分析。
4. **后验解释性分析相对欠缺。** 许多研究关注单次优化结果，却较少通过参数扰动、权重扫描等方式系统展示模型结论对关键参数变化的敏感程度。

基于上述不足，本文以新能源园区为研究对象，围绕微电网—电动车—建筑系统的协同调度问题展开研究。在问题一中，建立不考虑预测误差的确定性协同调度模型，统一协调光伏、固定储能、电动车聚合充放电及建筑柔性负荷，并与非协同策略进行对比；在问题二中，进一步将固定储能与 EV 电池寿命损耗纳入优化目标，构建兼顾运行成本与 ჯანმრთ损耗的调度模型，并通过权重扫描与灵敏度分析考察经济性与寿命之间的折中关系。本文的工作有助于为园区多资源协同调度提供更加统一、可解释且具有工程现实性的优化建模思路。
