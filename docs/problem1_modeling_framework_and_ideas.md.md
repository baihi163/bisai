# 问题一：园区微电网协同调度建模思路与框架设计

## 1. 核心问题剖析
问题一的核心在于实现园区能源系统从传统的“被动响应供电”向“源-网-荷-储-车”**主动协同运行**的转变。在完美预测（不考虑预测误差）的条件下，我们需要在长达 7 天（15分钟分辨率，共 672 个时段）的调度周期内，统筹处理以下核心矛盾：
1. **供需在时间上的错配**：光伏出力集中在白天且存在波动，而建筑负荷与 EV 充电需求可能集中在早晚高峰。
2. **多灵活性资源的竞争与互补**：固定储能、EV 电池（V2B）、建筑柔性负荷均具备“移峰填谷”的能力，如何分配它们的工作比例以实现全局经济性最优是关键。
3. **短期效益与长期损耗的权衡**：频繁调用储能和 EV 放电虽能减少当期购电成本，但会加速电池老化。

因此，本问题的本质是一个**多时段耦合的混合整数线性规划（MILP）问题**。

## 2. 整体建模框架构建
为了清晰地回答问题一，我们采取**“一主一副，对比印证”**的建模框架：
*   **主模型（协同优化调度模型）**：以上帝视角（全局最优）出发，将所有可调控资源纳入同一个目标函数下，通过数学规划求解器（如 Gurobi/CBC）寻找全周期的理论最优调度轨迹。
*   **副模型（非协同规则基准模型，Baseline）**：模拟现实中缺乏中央协调的园区现状（如：车主到站即充、储能仅按固定时段充放、负荷刚性不可调），作为对照组。

通过主副模型的 KPI 对比（如总成本、弃光率、峰值负荷），直观且定量地证明“协同调度”的巨大价值。

## 3. 关键子系统建模思路与亮点

### 3.1 电动汽车（EV）：从“无序盲盒”到“移动充电宝”
*   **降维处理思路**：EV 的接入具有极强的时空离散性。我们在建模时，**仅在 EV 处于“在站（Parked）”状态时为其分配决策变量**，这极大地压缩了求解空间。
*   **V2B 机制设计**：对于具备双向充放电资格的车辆，将其视为“移动储能”。模型不仅要保证其离站时达到目标电量，还要在光伏过剩时吸纳电能，在电网限电或高价时段向园区反向放电，赚取差价或填补缺口。

### 3.2 建筑柔性负荷：虚拟储能的“能量弹簧”
*   **平移与反弹机制**：将建筑负荷分为基础负荷与可调负荷。我们引入了“虚拟能量池（Backlog）”的概念来追踪被平移的负荷。
*   **能量守恒**：为了防止模型“作弊”（即无限期推迟负荷），我们设定了严格的能量中性约束，即“欠下的负荷必须在调度周期内偿还”，且考虑了负荷恢复时的反弹放大效应（Rebound Factor）。

### 3.3 固定储能（ESS）：跨时段套利的核心枢纽
*   **状态连续性**：通过建立 SOC（荷电状态）的跨时段递推方程，将离散的 672 个时段串联起来。
*   **充放电互斥**：引入 0-1 整数变量，严格约束储能在同一物理时段内不能同时充放电，保证模型的物理现实意义。

### 3.4 异常事件的隐式鲁棒处理
题目数据中隐含了暴雨导致光伏骤降、电网临时限电等“特殊事件”。我们在确定性建模中，直接将这些限制转化为**时变边界条件**（如时变的 $P_{import\_max, t}$）。协同模型能够利用其“全局视野”，在限电事件发生前提前蓄电，从而在不增加复杂随机规划的前提下，展现出优异的抗风险能力。

## 4. 目标函数的设计逻辑
我们将多维度的诉求统一折算为**经济成本**，构建了综合目标函数：
$$ \min \text{Total Cost} = \text{电网交互净成本} + \text{碳排放惩罚} + \text{弃光惩罚} + \text{电池退化成本} + \text{柔性调节补偿} $$

*   **引入退化成本的深意**：虽然问题一主要关注确定性调度，但我们在目标函数中预先以线性折旧的方式引入了储能和 EV 的退化成本。这不仅防止了求解器给出“疯狂充放电”的极端解，也为**问题二（深入研究寿命衰减模型）**做好了平滑的铺垫。

## 5. 预期对比分析维度 (协同 vs 非协同)
在论文的结果分析部分，我们将从以下几个维度展开论述：
1.  **经济性跃升**：协同模型通过“低充高放”和“光伏最大化利用”，总运行成本将显著低于 Baseline。
2.  **电网友好度**：Baseline 容易出现“EV集中充电与建筑晚高峰重叠”的“峰上加峰”现象；而协同模型能有效削峰填谷，降低最大购电功率（Peak Demand）。
3.  **新能源消纳**：协同模型能主动引导 EV 和储能在光伏大发时段充电，大幅降低弃光率（Curtailment Penalty）。
