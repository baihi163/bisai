# 确定性协同调度优化模型

**适用题目**：认证杯 B 题 第一阶段 问题 1  

本文档给出在**不考虑预测误差**条件下，基于附件给定的 **7 天、15 分钟分辨率**数据，对园区微电网—电动车—建筑系统进行**确定性协同调度**的联合优化模型。该模型为问题 1 的**主模型**（协同优化），与后续**非协同 baseline**在决策结构、信息利用方式与目标耦合关系上形成对照。

---

## 1. 模型定位与建模目标

问题 1 要求在已知未来各时段可再生能源出力、负荷与电价等信息的理想化设定下，构造一个**多时段、确定性、联合优化**的调度模型。园区在电气上等效为**单母线**结构：不建立支路潮流与网损模型，仅对有功功率进行逐时段平衡与资源协调。

建模目标可概括为：在物理可行域与运行规则约束下，通过统一优化外网购售电、储能充放电、光伏分配、电动车充放电（含具备资格的 V2B 放电）、建筑可调负荷及弃光/缺电等决策，使全调度时域内的**总运行成本**（或等价经济指标）达到最优。该优化在**整个时域 $T$ 上一次性求解**，而非按时段滚动采用固定启发式规则递推。

---

## 2. 协同调度的核心思想

**协同**在本模型中的含义是：光伏、固定储能、电动车（会话约束下）、建筑可调负荷与外购/上网功率**在同一目标函数与同一套功率平衡方程下被联合求解**；各子系统的可行域通过能量守恒、设备上下限与业务规则**显式耦合**，任一子系统的决策会改变其他子系统的最优边界。

具体而言：

- **源–网–荷–储–车**在同一优化问题中联动：购售电价格、光伏可用出力、储能 SOC、EV 到站/离站窗口与电量需求、建筑柔性边界共同决定各时段功率分配。
- **V2B（或车对园区）反向供电**仅对具备资格且在站车辆开放，其放电功率与储能放电、外售电、本地负荷共同竞争光伏剩余与购电空间，体现**多灵活性资源在统一目标下的替代与互补**。
- **建筑可调负荷**通过调整量进入功率平衡与成本项，使负荷侧与供给侧在同一框架下协调，而非事后修正负荷曲线。
- **全时域联合优化**：未来 $N$ 个时段的决策由单个数学规划一并确定，从而可利用**跨时段耦合**（储能 SOC、EV 会话电量、可调负荷累积效应等），区别于逐步、短视的规则型 baseline。

---

## 3. 基本假设

1. **确定性**：附件给出的 7 天、$\Delta t$ 分辨率数据被视为真实值，不考虑预测误差与不确定性集合。
2. **单母线等效**：园区内部网络聚合为单节点，**不计线路潮流、电压与无功**，只保证有功功率平衡。
3. **时间离散**：调度时段集合为 $T=\{1,\ldots,N\}$，步长 $\Delta t$（如 15 min），所有功率量为该时段内的**平均有功功率**（kW 或 MW），能量由功率对时间积分得到。
4. **建筑负荷分解**：各时段建筑用电需求由**不可调（原生）部分**与**可调部分**组成；可调部分通过**调整量**相对给定基准曲线偏移，并受上下界与可选爬坡等约束。
5. **电动车会话级行为**：每辆车（或聚合会话）具有到站/离站时刻、到站电量、离站最低需求电量及最大充放电功率；**仅在停车时段**允许充/放电；**仅具备 V2B 资格者**允许放电（V2B 功率非负）。
6. **弃光与缺电**：光伏出力可在本地消纳、储能充电、EV 充电、外送（售电）之间分配；超出接纳能力部分记为**弃光**。若模型允许切负荷，则用**未满足负荷**变量并施以惩罚（可选）。
7. **电网接口**：外购电与外售电功率受**时变上限**约束；必要时可引入购/售**互斥**（见约束），以避免同时大额购售的物理或合同限制。

## 4. 符号定义

### 4.1 集合与索引

| 符号 | 含义 |
|------|------|
| $T=\{1,\ldots,N\}$ | 调度时段索引集合 |
| $t\in T$ | 时段索引 |
| $\Delta t$ | 时间步长（h） |
| $\mathcal{B}$ | 建筑（或负荷节点）集合，单建筑时可退化为单元素 |
| $\mathcal{V}$ | 电动车会话（或车辆）集合 |
| $\mathcal{V}^{\mathrm{V2B}}\subseteq \mathcal{V}$ | 允许 V2B 放电的会话子集 |

### 4.2 已知参数（由附件或预处理给定）

| 符号 | 含义 |
|------|------|
| $P^{\mathrm{pv,avail}}_t$ | 时段 $t$ 光伏可用出力上界 |
| $P^{\mathrm{imp,max}}_t,\,P^{\mathrm{exp,max}}_t$ | 时段 $t$ 外购电、外售电功率上限 |
| $\lambda^{\mathrm{buy}}_t,\,\lambda^{\mathrm{sell}}_t$ | 时段 $t$ 购电、售电电价（或等价净费用系数） |
| $c^{\mathrm{curt}}_t$ | 弃光惩罚系数 |
| $c^{\mathrm{flex}}_{b,t}$ | 建筑 $b$ 在 $t$ 的柔性调节单位代价 |
| $c^{\mathrm{uns}}_t$ | 未满足负荷惩罚系数（若启用） |
| $P^{\mathrm{base}}_{b,t}$ | 建筑 $b$ 在 $t$ 的原生（基准）负荷功率 |
| $\underline{P}^{\mathrm{flex}}_{b,t},\,\overline{P}^{\mathrm{flex}}_{b,t}$ | 可调负荷调整量的下、上界 |
| 对会话 $v\in\mathcal{V}$： | |
| $t^{\mathrm{arr}}_v,\,t^{\mathrm{dep}}_v$ | 到站、离站时段索引（含端点约定见下） |
| $E^{\mathrm{arr}}_v,\,E^{\mathrm{req}}_v$ | 到站能量、离站最低需求能量 |
| $\overline{P}^{\mathrm{ch}}_v,\,\overline{P}^{\mathrm{dis}}_v$ | 最大充电、放电功率（V2B 仅 $v\in\mathcal{V}^{\mathrm{V2B}}$ 时 $\overline{P}^{\mathrm{dis}}_v>0$） |
| $\eta^{\mathrm{ch}}_{\mathrm{ess}},\,\eta^{\mathrm{dis}}_{\mathrm{ess}}$ | 固定储能充、放电效率 |
| $\overline{P}^{\mathrm{ch}}_{\mathrm{ess}},\,\overline{P}^{\mathrm{dis}}_{\mathrm{ess}}$ | 固定储能最大充、放电功率 |
| $\underline{E}_{\mathrm{ess}},\,\overline{E}_{\mathrm{ess}}$ | 储能能量下、上界 |
| $E_{\mathrm{ess},0}$ | 调度初始储能能量 |

**会话活跃性**：定义指示参数 $\chi_{v,t}\in\{0,1\}$：若时段 $t$ 内会话 $v$ 处于在站停车状态则 $\chi_{v,t}=1$，否则为 $0$。具体区间规则可与附件一致，例如 $t\in\{t^{\mathrm{arr}}_v,\ldots,t^{\mathrm{dep}}_v - 1\}$ 或闭区间，全文保持一致即可。

### 4.3 辅助：建筑总负荷与光伏分配

- 建筑 $b$ 在 $t$ 的**实际用电功率**记为 $P^{\mathrm{bld}}_{b,t}$，由基准与调整量构成（见决策变量）。
- 光伏出力在**本地负荷、储能充电、EV 充电、外售、弃光**之间分配，其和等于可用光伏减弃光。

## 5. 决策变量

对 $\forall t\in T$：

| 变量 | 含义 |
|------|------|
| $P^{\mathrm{imp}}_t$ | 外网购电功率（$\geq 0$） |
| $P^{\mathrm{exp}}_t$ | 外网上网/售电功率（$\geq 0$） |
| $P^{\mathrm{ch}}_{\mathrm{ess},t}$ | 固定储能充电功率 |
| $P^{\mathrm{dis}}_{\mathrm{ess},t}$ | 固定储能放电功率 |
| $P^{\mathrm{curt}}_t$ | 弃光功率 |
| $P^{\mathrm{uns}}_t$ | 未满足负荷功率（可选，$\geq 0$） |

对建筑 $b$、时段 $t$：

| 变量 | 含义 |
|------|------|
| $\Delta P^{\mathrm{flex}}_{b,t}$ | 可调负荷相对基准的调整量（可正可负，依符号约定） |

对会话 $v$、时段 $t$：

| 变量 | 含义 |
|------|------|
| $P^{\mathrm{ev,ch}}_{v,t}$ | EV 充电功率 |
| $P^{\mathrm{ev,dis}}_{v,t}$ | EV 放电功率（仅 $v\in\mathcal{V}^{\mathrm{V2B}}$ 允许非零） |

储能能量：

| 变量 | 含义 |
|------|------|
| $E_{\mathrm{ess},t}$ | 时段 $t$ 末（或 $t$ 内表征）储能能量 |

**可选二进制变量**（用于充放电互斥等）：

| 变量 | 含义 |
|------|------|
| $u^{\mathrm{ess}}_t\in\{0,1\}$ | 固定储能在 $t$ 是否处于充电模式（若采用互斥建模） |
| $u^{\mathrm{grid}}_t\in\{0,1\}$ | 电网在 $t$ 是否允许购电（若购售互斥） |

若采用**聚合 EV**而非会话级，可将 $\mathcal{V}$ 换为聚合段，并将会话约束换为聚合能量与功率包络；本文按**会话级**优先叙述。

## 6. 目标函数

采用**总运行成本最小**（收益以负号计入）。典型形式为：

$$
\min\ \sum_{t\in T} \Big[
\lambda^{\mathrm{buy}}_t\, P^{\mathrm{imp}}_t
- \lambda^{\mathrm{sell}}_t\, P^{\mathrm{exp}}_t
+ c^{\mathrm{curt}}_t\, P^{\mathrm{curt}}_t
+ \sum_{b\in\mathcal{B}} c^{\mathrm{flex}}_{b,t}\,\phi_{b,t}\bigl(\Delta P^{\mathrm{flex}}_{b,t}\bigr)
+ c^{\mathrm{uns}}_t\, P^{\mathrm{uns}}_t
\Big]\,\Delta t .
$$

其中 $\phi_{b,t}(\cdot)$ 为柔性代价函数，常取绝对值或二次型：

- **线性**：$\phi_{b,t}(x)=|x|$，可线性化为辅助变量与约束；
- **二次**：$\phi_{b,t}(x)=x^2$（凸二次规划）。

**经济含义**：购电增加成本，售电抵扣成本；弃光、柔性偏离与缺电均增加惩罚或代价，从而在统一目标下权衡光伏利用、电网交易、储能/EV 灵活性与用户体验。

## 7. 约束条件

### 7.1 建筑负荷建模

建筑 $b$ 在 $t$ 的实际用电功率为基准与调整量之和：

$$
P^{\mathrm{bld}}_{b,t} = P^{\mathrm{base}}_{b,t} + \Delta P^{\mathrm{flex}}_{b,t}.
$$

调整量满足（示例）盒约束与可选爬坡：

$$
\underline{P}^{\mathrm{flex}}_{b,t} \leq \Delta P^{\mathrm{flex}}_{b,t} \leq \overline{P}^{\mathrm{flex}}_{b,t}
$$

$$
\bigl|\Delta P^{\mathrm{flex}}_{b,t}-\Delta P^{\mathrm{flex}}_{b,t-1}\bigr| \leq \Delta P^{\mathrm{flex,ramp}}_{b,t}\quad (t\geq 2).
$$

若不需要爬坡，可省略后式。$\Delta P^{\mathrm{flex}}_{b,t}$ 进入功率平衡与目标中的柔性项，从而**建筑不再被当作固定外生负荷**。

### 7.2 固定储能

充放电功率非负且有上界：

$$
0 \leq P^{\mathrm{ch}}_{\mathrm{ess},t} \leq \overline{P}^{\mathrm{ch}}_{\mathrm{ess}},\qquad
0 \leq P^{\mathrm{dis}}_{\mathrm{ess},t} \leq \overline{P}^{\mathrm{dis}}_{\mathrm{ess}}.
$$

**SOC（能量）递推**（时段末能量）：

$$
E_{\mathrm{ess},t} = E_{\mathrm{ess},t-1}
+ \eta^{\mathrm{ch}}_{\mathrm{ess}}\,P^{\mathrm{ch}}_{\mathrm{ess},t}\,\Delta t
- \frac{1}{\eta^{\mathrm{dis}}_{\mathrm{ess}}}\,P^{\mathrm{dis}}_{\mathrm{ess},t}\,\Delta t,
$$

其中 $E_{\mathrm{ess},0}$ 给定，$t=1$ 时上式中 $E_{\mathrm{ess},0}$ 为初值。

能量边界：

$$
\underline{E}_{\mathrm{ess}} \leq E_{\mathrm{ess},t} \leq \overline{E}_{\mathrm{ess}},\quad \forall t.
$$

**充放电互斥（可选）**：引入 $u^{\mathrm{ess}}_t\in\{0,1\}$，

$$
0 \leq P^{\mathrm{ch}}_{\mathrm{ess},t} \leq u^{\mathrm{ess}}_t\,\overline{P}^{\mathrm{ch}}_{\mathrm{ess}},\qquad
0 \leq P^{\mathrm{dis}}_{\mathrm{ess},t} \leq (1-u^{\mathrm{ess}}_t)\,\overline{P}^{\mathrm{dis}}_{\mathrm{ess}}.
$$

若允许同时充放（物理上常不允许），则可不用互斥。

### 7.3 电动车会话级充放电

对 $\forall v\in\mathcal{V},\,t\in T$：

**仅在站可充放**：

$$
0 \leq P^{\mathrm{ev,ch}}_{v,t} \leq \chi_{v,t}\,\overline{P}^{\mathrm{ch}}_v,\qquad
0 \leq P^{\mathrm{ev,dis}}_{v,t} \leq \chi_{v,t}\,\overline{P}^{\mathrm{dis}}_v.
$$

**V2B 资格**：若 $v\notin\mathcal{V}^{\mathrm{V2B}}$，则

$$
P^{\mathrm{ev,dis}}_{v,t} = 0.
$$

**能量递推**（会话 $v$ 的能量状态 $E_{v,t}$）：

$$
E_{v,t} = E_{v,t-1}
+ \eta^{\mathrm{ch}}_{v}\,P^{\mathrm{ev,ch}}_{v,t}\,\Delta t
- \frac{1}{\eta^{\mathrm{dis}}_{v}}\,P^{\mathrm{ev,dis}}_{v,t}\,\Delta t
$$

（效率 $\eta^{\mathrm{ch}}_{v},\eta^{\mathrm{dis}}_{v}$ 为参数；若不计损耗可取 1。）初值 $E_{v,\,t^{\mathrm{arr}}_v - 1}=E^{\mathrm{arr}}_v$（按索引约定对齐）。

**离站最低需求**（在 $t^{\mathrm{dep}}_v$ 离开前末段能量满足）：

$$
E_{v,\,t^{\mathrm{dep}}_v - 1} \geq E^{\mathrm{req}}_v
$$

（具体对齐 $t^{\mathrm{dep}}_v$ 为离开时段起点还是终点时，能量不等式写法与 $\chi_{v,t}$ 一致即可。）

**非“到站即满充”**：上述约束仅强制**离站前达到需求下限**，不在到站瞬间强制充满，从而避免将 EV 简化为无会话结构的“到站即充”模型。

### 7.4 电网购售电

$$
0 \leq P^{\mathrm{imp}}_t \leq P^{\mathrm{imp,max}}_t,\qquad
0 \leq P^{\mathrm{exp}}_t \leq P^{\mathrm{exp,max}}_t.
$$

**购售互斥（可选）**：如存在 $u^{\mathrm{grid}}_t\in\{0,1\}$，

$$
P^{\mathrm{imp}}_t \leq u^{\mathrm{grid}}_t\,P^{\mathrm{imp,max}}_t,\qquad
P^{\mathrm{exp}}_t \leq (1-u^{\mathrm{grid}}_t)\,P^{\mathrm{exp,max}}_t.
$$

### 7.5 光伏：可用上限与分配平衡

光伏出力扣弃光后参与分配。记 $P^{\mathrm{pv,use}}_t = P^{\mathrm{pv,avail}}_t - P^{\mathrm{curt}}_t$，则

$$
0 \leq P^{\mathrm{curt}}_t \leq P^{\mathrm{pv,avail}}_t,
$$

且 $P^{\mathrm{pv,use}}_t$ 在**本地直接供负荷、对储能充电、对 EV 充电、对外售电**之间的分配满足（见下节总平衡）。若需显式写出“光伏支路”，可引入非负分配变量 $P^{\mathrm{pv}\rightarrow\mathrm{ld},t}$、$P^{\mathrm{pv}\rightarrow\mathrm{ess},t}$ 等，使它们的和等于 $P^{\mathrm{pv,use}}_t$；在单母线模型中，也常由单一平衡式直接隐含。

### 7.6 全系统有功功率平衡（完整）

单母线有功平衡要求：外来购电与本地发电（光伏净利用、储能放电、EV 放电）之和，等于外送、充电类负荷、原生+建筑用电、未满足部分等。采用**供应 = 需求** 形式之一如下：

$$
P^{\mathrm{imp}}_t + \bigl(P^{\mathrm{pv,avail}}_t - P^{\mathrm{curt}}_t\bigr)
+ P^{\mathrm{dis}}_{\mathrm{ess},t} + \sum_{v\in\mathcal{V}^{\mathrm{V2B}}} P^{\mathrm{ev,dis}}_{v,t}
=
P^{\mathrm{exp}}_t
+ P^{\mathrm{ch}}_{\mathrm{ess},t}
+ \sum_{v\in\mathcal{V}} P^{\mathrm{ev,ch}}_{v,t}
+ \sum_{b\in\mathcal{B}} P^{\mathrm{bld}}_{b,t}
+ P^{\mathrm{uns}}_t.
$$

**说明**：$\sum_{b} P^{\mathrm{bld}}_{b,t}$ 已包含**原生负荷与可调调整**；$P^{\mathrm{uns}}_t$ 为可选松弛，若不允许缺电则令 $P^{\mathrm{uns}}_t\equiv 0$ 并删去该项。若存在仅由光伏直供的支路，可将 $\bigl(P^{\mathrm{pv,avail}}_t - P^{\mathrm{curt}}_t\bigr)$ 拆分为显式流；在闭合形式下，上式已保证光伏在本地消纳、储能、EV、售电与弃光之间的**统一平衡**（售电与弃光通过 $P^{\mathrm{exp}}_t$、$P^{\mathrm{curt}}_t$ 与充电负荷共同体现）。

## 8. 模型性质说明

- **确定性多时段优化**：数据已知，形式为有限维数学规划；若含二进制变量（互斥）为 **MILP/MIQCP**，否则在线性/二次目标与线性约束下为 **LP/QP**。
- **耦合结构**：储能与 EV 能量状态跨时段耦合；建筑柔性可含爬坡；购售电价时变使目标跨时段耦合。
- **单母线**：无 DC/AC 潮流，**不可**用于支路过载或电压分析；若赛题后续要求网络安全，需另增约束或二层模型。
- **规模**：$N$ 为 7 天×每昼夜时段数（如 $\Delta t=15$ min 时 $N=7\times 96$）；会话数 $|\mathcal{V}|$ 较大时可采用聚合或列生成等，属算法实现层面，不改变协同建模本质。

## 9. 与非协同 baseline 的区别

| 维度 | 本模型（协同调度） | 典型非协同 baseline |
|------|-------------------|---------------------|
| 决策方式 | 全时域联合优化，单目标（或加权）下统一求解 | 分模块、分优先级或按时段启发式，逐步固定决策 |
| 建筑负荷 | $\Delta P^{\mathrm{flex}}_{b,t}$ 为决策，参与平衡与代价 | 常固定为预测值或事后调整 |
| EV | 会话级到站/离站、V2B 资格与能量约束 | 常简化为恒定充电或不受会话约束的功率上界 |
| 储能与光伏 | 与 EV、建筑、电网在同一平衡式中协调 | 常与 EV 解耦或按规则先储后放 |
| 购售电 | 受时变上限，可与互斥约束结合 | 常仅购电或简单限额 |
| 目标 | 显式经济成本+惩罚的权衡 | 常无全局最优，仅可行或单指标 |

---

## 10. 小结

本文给出了问题 1 在确定性假设下的**园区微电网—电动车—建筑**协同调度模型：时间集合 $T=\{1,\ldots,N\}$，步长 $\Delta t$；单母线有功平衡；建筑负荷分解为原生与可调；固定储能与 EV 会话（含 V2B 子集）能量递推与在站约束；光伏弃光与分配统一进入平衡；电网购售电带时变上限及可选互斥；目标为购电、售电、弃光、柔性及可选缺电惩罚的时域和。该模型为后续与**非协同 baseline**对比提供了明确的“联合优化 + 多资源耦合”参照。

---

## 11. 本模型为何属于协同调度模型

**第一，统一目标与统一可行域。** 光伏、固定储能、EV（含 V2B）、建筑可调负荷与外购/售电不在各自独立子问题中分别“算完再拼接”，而是在**同一目标函数**（总运行成本最小）与**同一功率平衡方程**下同时可行、同时最优。

**第二，侧向耦合显式。** 建筑 $\Delta P^{\mathrm{flex}}_{b,t}$ 改变总负荷，从而影响购电、弃光与储能/EV 的边际价值；EV 放电与储能放电、外售电在供应侧并列，存在**替代关系**；购售电价时变使跨时段转移（储能、EV 能量时移）具有经济意义。这些耦合在 baseline 的分步逻辑中通常被弱化或忽略。

**第三，跨时段最优而非单步最优。** 储能 SOC、EV 会话电量、柔性爬坡等约束连接多个时段，只有**全时域联合优化**才能充分利用“低谷充电、高峰放电/少购电”等时间套利；规则递推或单时段贪心一般无法保证全局最优。

**第四，与 baseline 的对照点清晰。** Baseline 往往将建筑固定、将 EV 简化或采用非联合规则；本模型明确保留**建筑可调**、**会话级 EV**（避免“到站即充”式过度简化）、**V2B 资格**与**全时域联合决策**，因此“协同”一词指**优化层面的联合性**，而非仅设备物理上的并网。
