# 数据说明

## 文件作用与字段说明

### `timeseries_15min.csv`
- **作用**：主时序驱动数据，提供负荷、光伏、电价、碳强度与电网交换约束（15 分钟粒度）。
- **主要字段**：
  - `timestamp`：时间戳（2025-07-14 00:00:00 至 2025-07-20 23:45:00）
  - `ambient_temp_c`：环境温度（degC）
  - `solar_irradiance_wm2`：太阳辐照（W/m2）
  - `office_building_kw` / `wet_lab_kw` / `teaching_center_kw`：各建筑原生负荷（kW）
  - `total_native_load_kw`：三类建筑原生负荷总和（kW）
  - `pv_available_kw`：可利用光伏出力上限（kW）
  - `grid_buy_price_cny_per_kwh` / `grid_sell_price_cny_per_kwh`：购售电电价（CNY/kWh）
  - `grid_carbon_kg_per_kwh`：电网碳强度（kgCO2/kWh）
  - `grid_import_limit_kw` / `grid_export_limit_kw`：购电/反送功率上限（kW）
- **B 题建模用途**：
  - 构建功率平衡约束（负荷、光伏、电网交换）
  - 计算经济目标（购售电成本）
  - 计算低碳目标（碳排放）
  - 形成分时段边界（电网进出功率限值）

### `asset_parameters.csv`
- **作用**：固定资产和系统参数字典（储能、光伏、充电设施、时间步长等）。
- **主要字段**：
  - `parameter`：参数名
  - `value`：参数值（数值）
  - `note`：参数说明
- **B 题建模用途**：
  - 固定储能（ESS）容量、功率、效率、初始电量等约束
  - 充电基础设施能力上限
  - 时间步长（与离散化一致性）

### `ev_sessions.csv`
- **作用**：电动车会话级数据（到离站时间、充放电能力、离站需求）。
- **主要字段**：
  - `session_id`：会话 ID
  - `ev_type`：车型
  - `arrival_time` / `departure_time`：到离站时间
  - `battery_capacity_kwh`：电池容量
  - `initial_energy_kwh`：到站初始电量
  - `required_energy_at_departure_kwh`：离站最低需求电量
  - `max_charge_power_kw` / `max_discharge_power_kw`：会话充/放电功率上限
  - `v2b_allowed`：是否允许车到楼放电（0/1）
  - `degradation_cost_cny_per_kwh_throughput`：电池吞吐折旧成本
- **B 题建模用途**：
  - 构建 EV 群体可用容量与可调功率边界
  - 建立到离站边界条件（初始能量流入与离站能量需求）
  - V2B 约束与寿命成本项

### `flexible_load_parameters.csv`
- **作用**：柔性负荷参数（按建筑负荷块）。
- **主要字段**：
  - `load_block`：负荷块名称
  - `noninterruptible_share`：不可中断比例
  - `max_shiftable_kw`：可转移功率上限
  - `max_sheddable_kw`：可削减功率上限
  - `rebound_factor`：反弹系数
  - `penalty_cny_per_kwh_not_served`：未服务惩罚
- **B 题建模用途**：
  - 建立需求响应可调区间约束
  - 对削减/未满足负荷引入惩罚成本
  - 建立移峰后的反弹补偿关系

### `daily_summary.csv`
- **作用**：按日汇总（峰值与日电量），用于快速核验规模。
- **主要字段**：
  - `date`：日期
  - `load_peak` / `load_energy_kwh`：负荷峰值与日负荷电量
  - `pv_peak` / `pv_energy_kwh`：光伏峰值与日光伏电量
- **B 题建模用途**：
  - 与时序积分结果交叉验证，检查数据与模型输出合理性

### `ev_summary_stats.csv`
- **作用**：EV 数据统计摘要（会话数、均值等）。
- **主要字段**：
  - `metric`：统计指标名
  - `value`：指标值
- **B 题建模用途**：
  - 用于设定参数先验、检查样本规模和基线工况

### `scenario_notes.csv`
- **作用**：场景文字说明与配置项。
- **主要字段**：
  - `item`：场景项
  - `value`：场景值/描述
- **B 题建模用途**：
  - 解释异常时段、约束触发背景及情景假设

### `README.txt`
- **作用**：数据集总说明（时间范围、字段语义、使用建议）。
- **B 题建模用途**：
  - 作为预处理和建模前的语义参考文档

## 可能的数据风险点

- 时间戳风险：`timestamp`、`arrival_time`、`departure_time` 可能存在时区/格式不一致或非 15 分钟对齐。
- 一致性风险：`total_native_load_kw` 可能与三类建筑负荷求和存在微小偏差。
- 物理边界风险：存在负值功率、超容量电量、`departure <= arrival` 等无效会话。
- 约束冲突风险：EV `v2b_allowed=0` 但放电功率非零；或初始电量大于容量、离站需求大于容量。
- 缺失与重复风险：关键字段缺失值、会话 ID 重复、时间序列重复或断档。
- 语义漂移风险：`scenario_notes.csv` 的文本项需要人工解释，难以直接结构化入模。
